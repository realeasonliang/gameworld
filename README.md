## 怎么玩 · How to Play

1. 把仓库下载 / 克隆到本地。
   *Download or clone the repo.*
2. 双击任意一个 `.html` 文件，用浏览器打开即可。
   *Double-click any `.html` file to open it in your browser.*
3. 每个游戏里都有操作说明（或按 `H` / 看屏幕按钮）。
   *Each game shows its own controls on screen.*

```bash
git clone git@github.com:realeasonliang/gameworld.git
```
